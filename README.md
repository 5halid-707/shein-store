# KMH Fashion — متجر KMH للأزياء

متجر إلكتروني عصري مستوحى من Shein، مبني بـ Next.js 16 + TypeScript + Tailwind CSS 4.

## 🛍️ المميزات

- ✅ واجهة عربية RTL كاملة
- ✅ تصميم متجاوب (Mobile-first)
- ✅ صفحة رئيسية بكاروسيل + فلاش سيل + وصل حديثاً + الأكثر مبيعاً
- ✅ صفحة منتج بمعرض صور + اختيار المقاس/اللون/الكمية + تقييمات
- ✅ سلة تسوق كاملة مع كود خصم
- ✅ صفحة دفع مع طرق دفع متعددة (مدى، Apple Pay، COD...)
- ✅ صفحة تأكيد الطلب
- ✅ صفحة فئات بفلاتر (مقاس، لون، سعر) وفرز
- ✅ قائمة مفضلة محفوظة محلياً
- ✅ شريط تنقل سفلي للموبايل
- ✅ Cart drawer جانبي
- ✅ إشعارات Toast

## 🚀 التشغيل

```bash
bun install
bun run dev      # تطوير (port 3001)
bun run build    # بناء للإنتاج
bun run start    # تشغيل الإنتاج
```
