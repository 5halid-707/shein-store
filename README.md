# SHEIN — متجر الأزياء العصرية (Clone للعرض)

متجر إلكتروني كامل مستوحى من موقع SHEIN (النسخة العربية) مبني بتقنية Next.js 16
لأغراض العرض والتطوير. التصميم مطابق بصرياً لموقع Shein: الشعار، الألوان،
البنية، وبطاقات المنتجات.

> ⚠️ **تنبيه**: هذا الموقع للعرض فقط ولا يقبل طلبات حقيقية. تصميم وتطوير
> **خالد الحربي**.

## المميزات

- شعار SHEIN بصيغة SVG (نسخة سوداء وبيضاء)
- شريط إعلانات متحرك (marquee) بنمط Shein
- Header مطابق لـ Shein: شعار + بحث + أيقونات (حساب/مفضلة/سلة) + قائمة فئات
- Hero carousel بصور أزياء حقيقية من Unsplash
- قسم Flash Sale مع عدّاد تنازلي
- بطاقات منتجات بنمط Shein (شارة خصم حمراء، ألوان، زر "أضف للسلة" عند المرور)
- صفحات منتج، فئة، سلة، دفع، تأكيد طلب، مفضلة، حساب
- 28 منتج بصور حقيقية من Unsplash عبر 6 فئات
- RTL عربي بالكامل + responsive (mobile-first)
- معلومات المالك في الفوتر: khalid-alharbi@zohomail.sa / +966 57 501 5019

## التقنيات

- Next.js 16 + TypeScript 5
- Tailwind CSS 4
- Framer Motion
- Lucide icons
- React Context (cart / wishlist / toast) + localStorage

## تطوير

```bash
bun install
bun run dev      # localhost:3001
bun run build
bun run start
```
